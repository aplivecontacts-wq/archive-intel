# Light Theme Update

Updates all remaining components to use light theme colors.

## Color Mapping:
- bg-black → bg-white or bg-gray-50
- text-emerald-400 → text-emerald-600/700
- text-emerald-100 → text-gray-900/800
- border-emerald-500/20 → border-emerald-200
- bg-emerald-950/20 → bg-white or bg-emerald-50
